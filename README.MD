## UDPServer and UDPClient
move to workplace and compile Server and Client
```aidl
    cd $PROJECT_PATH 
    ./build.sh
    cd bin/
```

use command bellow to start UDPServer
```aidl
    java -jar UDPServer.jar 
```

use command bellow to start UDPClient
```aidl
    java -jar UDPClient.jar

```
back to workplace
```aidl
    cd ../
```
